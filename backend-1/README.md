# OOP Chat Backend (TypeScript + Node.js) - OOP Refactor

This project is a refactored **Object-Oriented** backend for the "Individual project".
It implements domain models as classes, repositories, services and a ChatServer class.

How to run:
1. unzip and `cd` into project
2. `npm install`
3. For development: `npm run dev`
   Or build: `npm run build` and `npm start`

Notes:
- Uploaded avatars are stored under `/uploads/avatars`
- Database file: `data/database.sqlite`
- Server listens on port 8080
